Der Kosten- und Finanzierungsplan (KFP) ist neben der Projektbeschreibung das wichtigste Dokument jedes Förderantrags und das, an dem die meisten Anträge scheitern. Ein KFP ist keine Buchhaltung, sondern ein Argument: Er zeigt der Jury, dass ihr euer Projekt verstanden habt, realistisch plant und euch selbst nicht ausbeutet. Dieser Baustein vermittelt die Logik, das Vokabular und die Praxis der Kalkulation.

## Block 1: Anatomie eines KFP (45 Min.)

### Das Grundprinzip: Der Plan muss "aufgehen"

Ein KFP hat immer zwei Seiten:

- **Ausgaben** (Kosten): Was kostet das Projekt insgesamt?
- **Einnahmen** (Finanzierung): Woher kommt das Geld?

**Eiserne Regel: Ausgaben = Einnahmen.** Beide Seiten muessen sich exakt auf den Cent decken ("ausgeglichener KFP"). Die beantragte Foerdersumme ist eine Position auf der Einnahmenseite -- nicht das Gesamtbudget.

### Die Ausgabenseite -- typische Positionen

| Kategorie                                             | Beispiele                                                                                                                                 |
| ----------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| **Honorare (kuenstlerisch)**                          | Eigenes Honorar, Honorare fuer beteiligte Kuenstler*innen, Komposition, Performance                                                       |
| **Honorare / Dienstleistungen (nicht-kuenstlerisch)** | Technik-Crew, Produktionsleitung, Dokumentation, Uebersetzung, Grafik                                                                     |
| **KSK-Abgabe**                                        | Abgabe auf kuenstlerische Honorare, wenn ihr als Veranstalter*in/Traeger andere Kuenstler*innen beauftragt (2026: aktuellen Satz pruefen) |
| **Material & Produktion**                             | Baumaterial, Elektronik, Druck, Verbrauchsmaterial                                                                                        |
| **Technik**                                           | Miete (Beamer, Lautsprecher, Licht), Anschaffungen (Achtung: oft nur anteilig oder gar nicht foerderfaehig!)                              |
| **Raum**                                              | Miete fuer Ausstellungs-/Probenraum, Nebenkosten, Versicherung                                                                            |
| **Reise & Transport**                                 | Fahrten, Uebernachtung, Werktransport (Bundesreisekostengesetz als Orientierung)                                                          |
| **Oeffentlichkeitsarbeit**                            | Plakate, Flyer, Website, Anzeigen, Fotografie                                                                                             |
| **Dokumentation**                                     | Foto/Video der fertigen Arbeit                                                                                                            |
| (**Verwaltungskosten**)                               | oft pauschal max. 5-10 % der Gesamtkosten, je nach Foerderrichtlinie                                                                      |
| **Unvorhergesehenes**                                 | ueblich 5-10 % - manche Foerderer akzeptieren diese Position nicht, Richtlinie lesen!                                                     |
| Versicherung                                          | Projektbezogene Versicherung (z.B. Veranstaltungsversicherung)                                                                            |
| Personalkosten                                        | Aufsichtspersonal (Mindestlohn)                                                                                                           |

**Wichtig:** Jede Position muss nachvollziehbar kalkuliert sein. Statt "Material: 800 EUR" besser "Stahlrohr 6m (4x 45 EUR), Lautsprechertreiber (8x 35 EUR), ...". Im Antrag reicht oft die Summenebene, aber ihr muesst die Detailkalkulation in der Schublade haben - spaetestens beim Verwendungsnachweis braucht ihr sie.

### Die Einnahmenseite - woher kommt das Geld?

| Position                         | Was ist das?                                                                                                                                                                         |
| -------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Beantragte Foerderung**        | Die Summe, die ihr bei diesem Foerderer beantragt                                                                                                                                    |
| **Eigenmittel (bar)**            | Eigenes Geld, das ihr einbringt - viele Foerderer verlangen eine Quote (z.B. 10-20 %)                                                                                                |
| **Unbare Eigenleistungen**       | Eigene Arbeitszeit, eigene Technik, eigener Raum -- ohne Geldfluss. Wird auf beiden Seiten in gleicher Hoehe eingetragen. Nicht jeder Foerderer akzeptiert sie -- Richtlinie pruefen |
| **Drittmittel / Kofinanzierung** | Zusagen anderer Foerderer, Stiftungen, Sponsoring (Status angeben: beantragt / zugesagt)                                                                                             |
| **Eigene Einnahmen**             | Tickets, Kataloge, Teilnahmegebuehren, Verkaeufe, Editionen (Erwartet (Schätzung))                                                                                                   |
| **Spenden / Crowdfunding**       | realistisch ansetzen, nicht als Lueckenfueller schoenen                                                                                                                              |

**Unbare Eigenleistungen sind euer Freund:** Wer 100 Stunden eigene Arbeit zum BBK-Satz als unbare Eigenleistung eintraegt, erfuellt damit oft die Eigenanteils-Quote, ohne Geld in die Hand zu nehmen. Aber: Position muss auf der Ausgabenseite in gleicher Hoehe stehen, sonst geht der Plan nicht auf.

### Mini-Diskussion (5 Min.)
Warum verlangen Foerderer ueberhaupt Eigenmittel oder Kofinanzierung? (Risikoteilung, Ernsthaftigkeit, Hebelwirkung oeffentlicher Mittel)

## Block 2: Honorare - eure Arbeitszeit ist eine Kostenposition 

Der haeufigste und teuerste Fehler in studentischen Antraegen: **das eigene Honorar fehlt oder ist laecherlich niedrig.** Ein KFP ohne eigenes Honorar sagt der Jury: "Meine Arbeit ist nichts wert." Das ist weder strategisch klug noch foerderpolitisch erwuenscht.

### Der BBK Leitfaden Honorare

Der Bundesverband Bildender Kuenstler*innen empfiehlt seit 2023 fuer kuenstlerische Leistungen einen Satz von **netto 70 EUR pro Stunde** und gliedert kuenstlerische Arbeit in vier Phasen:

1. **Vorbereitung** -- Recherche, Konzeption, Planung
2. **Projektsteuerung** -- Organisation, Kommunikation, Verwaltung
3. **Umsetzung** -- Produktion, Auf-/Abbau
4. **Vermittlung** -- Kuenstler*innengespraech, Workshop, Fuehrung

Dazu kommt ggf. die **Ausstellungsverguetung** (Verguetung fuer das Bereitstellen von Werken) -- das ist eine eigene Position, kein Ersatz fuers Arbeitshonorar.

### Honoraruntergrenzen (HUG) - vom Appell zur Pflicht

Seit dem **1. Juli 2024** gelten verbindliche Honoraruntergrenzen fuer kuenstlerische Arbeit bei Foerderungen des Bundes (BKM), wenn der Bundesanteil 50 % uebersteigt. Foerderer wie die Kulturstiftung des Bundes pruefen das: Wer die HUG unterschreitet, muss nachzahlen, und der Betrag ist dann nicht zuwendungsfaehig. Auch Laender und Kommunen ziehen nach (NRW z.B. ab 2026 fuer alle Sparten). **Konsequenz:** Anstaendige Honorare zu kalkulieren ist keine Frechheit, sondern zunehmend Foerdervoraussetzung.

### Live-Demo (15 Min.)

Gemeinsam am Beamer: [Honorarrechner des BBK RLP](https://www.honorarrechner-bbkrlp.de/) -- ein fiktives Projekt durchklicken (z.B. Klanginstallation in einer Gruppenausstellung: 20 Std. Vorbereitung, 10 Std. Steuerung, 24 Std. Umsetzung, 4 Std. Vermittlung) und das Ergebnis als Honorarposition in einen KFP uebernehmen.

### KSK nicht vergessen

Wenn ihr als Projekttraeger andere selbststaendige Kuenstler*innen beauftragt, seid ihr ggf. **kuenstlersozialabgabepflichtig**. Die Abgabe (Prozentsatz auf die Honorarsumme, aktuellen Satz auf kuenstlersozialkasse.de pruefen) gehoert als eigene Position in den KFP -sonst zahlt ihr sie aus eigener Tasche. Meldepflichtig ist nicht nur das Honorar, sondern auch Aufwendungsersatz und Nebenleistungen.

## Block 3: Zuwendungsrecht-Basics - das Kleingedruckte (30 Min.)

Foerdergeld ist kein Geschenk, sondern eine **Zuwendung** mit Regeln. Die wichtigsten Begriffe, die ihr kennen muesst:

### Finanzierungsarten

| Art                         | Bedeutung                                                  | Konsequenz fuer euch                                                 |
| --------------------------- | ---------------------------------------------------------- | -------------------------------------------------------------------- |
| **Fehlbedarfsfinanzierung** | Foerderer deckt die Luecke zwischen Einnahmen und Ausgaben | Mehreinnahmen (z.B. mehr Tickets) reduzieren die Foerderung!         |
| **Anteilfinanzierung**      | Foerderer zahlt festen Prozentsatz der Gesamtkosten        | Sinken die Kosten, sinkt die Foerderung anteilig                     |
| **Festbetragsfinanzierung** | Fester Betrag, unabhaengig von Abweichungen                | am angenehmsten -- Mehreinnahmen bleiben euch (innerhalb der Regeln) |
| **Vollfinanzierung**        | Foerderer traegt 100 %                                     | selten, meist bei Stipendien                                         |

### Die wichtigsten Spielregeln

- **Vorzeitiger Massnahmebeginn:** Vor der Bewilligung darf das Projekt nicht begonnen haben (keine Vertraege, keine Bestellungen!) -- ausser ihr beantragt explizit den vorzeitigen Massnahmebeginn und er wird genehmigt.
- **Bewilligungszeitraum:** Nur Ausgaben innerhalb dieses Zeitraums sind foerderfaehig. Rechnungsdatum zaehlt.
- **Zuwendungsfaehige Kosten:** Nicht alles ist foerderbar -- haeufig ausgeschlossen: Anschaffungen/Investitionen (Technik kaufen statt mieten!), Bewirtung, laufende Betriebskosten, manchmal "Unvorhergesehenes". Immer die Richtlinie und die ANBest-P lesen.
- **Umwidmung:** Verschiebungen zwischen Positionen sind meist nur begrenzt erlaubt (oft bis 20 % je Position), darueber hinaus nur nach Ruecksprache mit dem Foerderer. Im Zweifel: vorher fragen, nicht hinterher beichten.
- **Doppelfoerderung:** Dieselbe Ausgabe darf nicht von zwei Foerderern bezahlt werden. Kofinanzierung ja, Doppelabrechnung nein.
- **Besserstellungsverbot:** Personal darf nicht besser bezahlt werden als vergleichbare Stellen im oeffentlichen Dienst (TV-L als Obergrenze). 
- **Wirtschaftlichkeit & Vergabe:** Bei groesseren Auftraegen Vergleichsangebote einholen (oft 3 Angebote ab bestimmten Summen). Kuenstlerische Leistungen sind davon i.d.R. ausgenommen, weil nicht vergleichbar.
- **Verwendungsnachweis:** Nach Projektende muesst ihr abrechnen -- **Sachbericht** (was ist passiert?) + **zahlenmaessiger Nachweis** (Ist-Zahlen gegen den Plan) + Belege aufbewahren (oft 5-10 Jahre). Wer den KFP von Anfang an sauber im Vault fuehrt, hat hier kaum Arbeit.

**Merksatz:** Der KFP, den ihr einreicht, ist ein Vertrag mit eurer Zukunft. Kalkuliert so, dass ihr in 12 Monaten noch dazu stehen koennt.

## Uebung 1: Der Fehler-KFP (20 Min., Gruppen a 2-3)

Verteilt den vorbereiteten fehlerhaften KFP (Vorlage unten als Kopiervorlage). Aufgabe: **Findet mindestens 8 Probleme.** Danach Aufloesung im Plenum.

Eingebaute Fehler (fuer die Seminarleitung):

1. Ausgaben und Einnahmen decken sich nicht (Differenz 340 EUR)
2. Kein eigenes Honorar kalkuliert
3. Fremdhonorar mit 15 EUR/Std. weit unter jeder Honoraruntergrenze
4. KSK-Abgabe fehlt trotz externer kuenstlerischer Honorare
5. Laptop-Neukauf als Position (Investition, i.d.R. nicht zuwendungsfaehig -- mieten oder anteilige Abschreibung)
6. "Sonstiges: 1.500 EUR" ohne jede Aufschluesselung
7. Unbare Eigenleistung nur auf der Einnahmenseite eingetragen
8. Drittmittel als "zugesagt" markiert, obwohl nur beantragt (im Begleittext erkennbar)
9. Projektbeginn laut Zeitplan vor der Jurysitzung (vorzeitiger Massnahmebeginn)
10. Erwartete Ticketeinnahmen unrealistisch hoch angesetzt, um die Luecke zu stopfen

> Tipp zur Vorbereitung: Den Fehler-KFP als eigene Datei `fehler_kfp.md` oder Tabelle ins Repo legen, damit die Gruppen ihn vor sich haben.

## Uebung 2: Eigener KFP (50 Min.)

Nehmt die **Projektbeschreibung aus der Uebung [Bewerbung & Foerderung](exercise5_bewerbungen_foerderung.md)** (oder ein reales Vorhaben) und erstellt dazu einen vollstaendigen KFP.

Vorgehen:

1. **Ausgaben brainstormen (10 Min.)** -- alles aufschreiben, was das Projekt kostet. Checkliste aus Block 1 nutzen.
2. **Eigenes Honorar kalkulieren (10 Min.)** -- Stunden je Arbeitsphase schaetzen, [Honorarrechner](https://www.honorarrechner-bbkrlp.de/) oder 70 EUR/Std. ansetzen.
3. **Einnahmenseite bauen (15 Min.)** -- Foerderung, Eigenmittel, unbare Eigenleistungen, realistische Einnahmen. Plan ausgleichen.
4. **Plausibilitaets-Check (15 Min.)** -- Selbstpruefung mit der Checkliste unten.

Den KFP als Markdown-Tabelle oder verlinkte Tabelle im Obsidian Vault unter `04_Bewerbungen/<Ausschreibung>/kfp.md` ablegen -- so waechst euer Bewerbungsarchiv weiter.

### Checkliste Plausibilitaet

- [ ] Ausgaben = Einnahmen, auf den Cent
- [ ] Eigenes Honorar enthalten und begruendbar (Stunden x Satz)
- [ ] Keine Pauschalposten ohne Aufschluesselung
- [ ] KSK bedacht (falls externe kuenstlerische Honorare)
- [ ] Keine reinen Anschaffungen (oder Richtlinie erlaubt sie explizit)
- [ ] Unbare Eigenleistungen auf beiden Seiten
- [ ] Status der Drittmittel ehrlich angegeben (beantragt/zugesagt)
- [ ] Puffer eingeplant (falls zulaessig)
- [ ] Alle Zahlen wuerde ich in 12 Monaten gegenueber dem Foerderer verteidigen koennen

## Jury-Check (15 Min., Paare)

Tauscht eure KFPs. Lest sie mit Jury-Brille:

- Wirkt die Kalkulation realistisch -- weder geschoent noch aufgeblasen?
- Versteht man jede Position ohne Nachfrage?
- Wuerde ich diesem Menschen 8.000 EUR anvertrauen?
- Was wuerde ich als Foerderer streichen oder hinterfragen?
