# Teaching Concept: "Clear your backlog"

**Seminar:** Clear your backlog
**Institution:** Kunsthochschule Kassel, Neue Medien
**Format:** Bi-weekly, 1 semester (approx. 14 weeks = 7 sessions)
**Duration per session:** 3 hours (180 min)
**Target group:** Art and design students, all levels
**Language:** German (materials bilingual DE/EN)

---

## Seminar Goals

By the end of the semester, each student will have:
- A structured Obsidian vault for managing their artistic/design practice
- Written work descriptions for their key pieces
- A written artist/designer statement (DE + EN)
- A professional CV tailored to their career stage
- Skills in professional communication (applications, emails, inquiries)
- Good documentation of their work (photo, video, screenshots)
- A tech rider for at least one work
- A working website or portfolio page online
- A portfolio PDF ready for applications
- An overview of the funding landscape (grants, residencies, prizes, festivals)
- An understanding of institutional contexts and how to navigate them
- A personal vision for their next professional steps

## Semester Timeline

### Session 1 -- Kickoff & Organization
**Week 1 | Theme: Getting started, setting up systems**

| Time | Activity | Format |
|------|----------|--------|
| 0:00-0:20 | Welcome, introductions, seminar overview | Plenary |
| 0:20-0:40 | "Backlog audit": Students list everything they've been meaning to do professionally (website, CV, statement, applications...) and prioritize | Individual + sharing |
| 0:40-1:10 | Introduction to Obsidian: Install, tour of features, why knowledge management matters for artists/designers | Demo + hands-on |
| 1:10-1:20 | Break | |
| 1:20-1:50 | Set up personal Obsidian vault from template (01_organize). Customize folder structure, fill in basic personal data | Hands-on |
| 1:50-2:20 | Exercise: Document one existing work using the work template (Titel der Arbeit.md). Add image to Footage folder | Individual |
| 2:20-2:40 | Show & tell: 2-3 volunteers share their documented work | Plenary |
| 2:40-3:00 | Homework briefing + Q&A | Plenary |

**Homework (due Session 2):**
- Document at least 3 works in the Obsidian vault using the template
- Collect all existing text fragments (old bios, application texts, etc.) into the vault
- Bring a laptop with Obsidian installed to every session

**Materials:** `01_organize/`

### Session 2 -- Writing About Your Work
**Week 3 | Theme: Describing artistic work clearly**

| Time | Activity | Format |
|------|----------|--------|
| 0:00-0:15 | Check-in: How did the documentation go? Questions about Obsidian? | Plenary |
| 0:15-0:45 | Exercise 1 -- Pair Presentations (`02_write/exercise1_works.md`): Students present one work to a partner (10-15 min each). Partner takes structured notes | Pairs |
| 0:45-1:15 | Writing phase: Based on partner's notes, write a short summary (3-4 sentences) and a longer text (1000-2000 characters) for the presented work | Individual |
| 1:15-1:25 | Break | |
| 1:25-1:55 | Feedback round: Read texts aloud. Group discusses: Does the text convey the work? What's vivid, what's vague? | Plenary |
| 1:55-2:30 | Revision: Incorporate feedback, refine texts. Store final versions in Obsidian vault under the respective work entry | Individual |
| 2:30-2:50 | Input: Common pitfalls in art writing -- jargon, passive voice, vagueness. Show good and bad examples | Lecture |
| 2:50-3:00 | Homework briefing | Plenary |

**Homework (due Session 3):**
- Write short + long descriptions for all documented works in the vault
- Read the Statement exercise (exercise2_statement.md) as preparation
- Bring physical writing materials (pen, paper, index cards) for cluster exercise

**Materials:** `02_write/exercise1_works.md`

| 1:15-1:25 | Break | |
| 1:25-2:00 | Writing phase: Draft artist/designer statement following the structure (entry/deepening/method/impact). First person, active, present tense | Individual |
| 2:00-2:15 | Voluntary readings + group feedback | Plenary |
| 2:15-2:45 | CV Workshop (`02_write/exercise3_cv.md`): Input on artistic CV vs. standard CV. Analyze 3-4 example CVs together | Lecture + Discussion |
| 2:45-3:00 | Start drafting CV in Obsidian vault. Discuss: What goes in, what stays out at your career stage? | Hands-on + Q&A |

**Homework (due Session 4):**
- Revise artist statement after letting it rest a few days
- Write statement in second language (EN or DE)
- Complete CV draft in Obsidian vault (both languages)
- Peer review: Exchange statement with one classmate for written feedback

**Materials:** `02_write/exercise2_statment.md`, `02_write/exercise3_cv.md`, `05_Portfolio/` (example PDFs)

## Cross-Navigation

All exercise and content files include a consistent navigation bar (blockquote format) linking to every other module. The current page is bold, all others are clickable relative links. This allows students to move freely between topics without returning to the README.

Navigation order follows the seminar's logical flow:
Vault organisieren > Werkbeschreibung > Statement > CV > Kommunikation > Dokumentation > Tech Rider > Website > Portfolio > Bewerbung & Foerderung > Institutionen > Leseliste
